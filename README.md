## ✨ Features

This portfolio highlights my **professional journey and front-end skills**:

- 🌟 **Responsive design:** Optimized for desktop, tablet, and mobile  
- 🎨 **Animated Hero section:** Professional introduction with social media links  
- 🖥️ **Desktop navbar:** Hover animations and smooth scrolling  
- 📱 **Fullscreen mobile menu:** Hamburger menu for mobile devices  
- 📂 **Projects showcase:** Interactive project cards with links and images (planned)  
- 🛠️ **Skills section:** Technical skills, frameworks, and tools (planned)  
- 🧑‍💼 **Professional journey:** About section highlighting education, experience, and achievements  
- 📬 **Contact section:** GitHub, LinkedIn, email  
- ⚡ **Planned enhancements:**  
  - Dark/light theme toggle  
  - Interactive project filters  
  - Animations and transitions for smoother UX  
  - Downloadable resume button

## 💡 Usage

Click navbar links to scroll to each section

On mobile, use the hamburger menu to open the fullscreen menu

Social media buttons redirect to GitHub, LinkedIn, or open your email client
